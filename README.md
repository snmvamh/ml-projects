# ML Projects

Проекты по машинному обучению. 

## Проекты
Готово, вот список:

- **PandasMatplotlibML.ipynb**: анализ и визуализация данных (pandas, matplotlib, seaborn)
- **LinearRegressionKNNML.ipynb**: линейная регрессия и KNN (scikit-learn, seaborn)
- **LinearModelsML.ipynb**: линейные модели (scikit-learn)
- **TreesRandomForestML.ipynb**: деревья и случайный лес (scikit-learn, matplotlib)
- **BacteriaML.ipynb**: классификация бактерий (scikit-learn, numpy)
- **StellarML.ipynb**: классификация астрономических объектов (scikit-learn, numpy)
- **PhonemesML.ipynb**: классификация фонем (scikit-learn, pandas)
- **RecognizerML.ipynb**: задача распознавания (scikit-learn, pandas)
- **GenomicsML.ipynb**: ML на геномных данных с эмбеддингами (scikit-learn, sentence-transformers)
- **AutoML.ipynb**: регрессия по данным об авто (scikit-learn, pandas)

## Стек

Python, NumPy, pandas, scikit-learn, matplotlib, seaborn.
